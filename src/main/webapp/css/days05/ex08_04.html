[플렉스 항목(item) 속성]
		1. flex-grow 
			- 나머지 플렉스 항목에 비해
			플렉스 항목이 얼마나 커질지 지정하는 속성
			기본값 : 0
			
		2. flex-shrink 속성
			나머지 플렉스 항목에 비해 얼마나 축소되는지 지정하는 속성
			기본값 : 1
			
		3. flex-basis 속성
			기초, 근거, 주성분
			플렉스 항목의 초기 너비를 설정
			
		4. flex 약식
			flex: flex-grow flex-shrink flex-basis
			
3

			
		5. align-self 속성 ex08_05.html
			선택한 플렉스 항목의 정렬 지정할 수 있다.
